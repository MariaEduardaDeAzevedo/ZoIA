Sobre

ZoIA é uma extensão para navegadores (atualmente em funcionamento apenas para Google Chrome) que injeta descrições de imagens em páginas Web.

Registrar

Para utilizar o ZoIA é necessário criar uma conta. Utilizamos isso apenar para controle técnico de uso do serviço, basta que cadastre um bom e-mail, para que você possa manter contato, caso necessário, e uma senha segura.
